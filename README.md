# Final project

# 기획의도

다양한 상황에서의 영화 추천 시스템 제작을 목표로 함.
혼밥을 하는 사람, 친구, 가족, 연인과 함께인 상황에서 영화를 더 빠르게 고를 수 있게 도와주는 서비스를 만들고자 함.

또한 다양한 추천 시스템(상황 별 추천, 랜덤 추천, 일반적인 인기순, 평점순, 장르 별 등)을 통해 사용자는 더 많은 추천을 받을 수 있도록 함
